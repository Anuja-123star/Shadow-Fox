Task 1 - Variables
Objective

This task demonstrates the use of variables, data types, reserved keywords, and arithmetic operations in Python.

Tasks Completed
Created a variable pi and stored the value 22/7.
Displayed the data type of the variable using the type() function.

Calculated the Simple Interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

Technologies Used
Python 3

How to Run
Open the project in any Python IDE (VS Code, PyCharm, etc.).
Run the main.py file.


Output

Displays the value and data type of pi.
Explains why 'for' cannot be used as a variable name.
Calculates and displays the Simple Interest.
